## Key Concepts

### LangChain LCEL
- Language Chain Expression Language
- Declarative way to build chains
- Composable and chainable components

### ReAct Pattern
- Reasoning and Acting
- Combines reasoning with tool use
- Iterative problem-solving approach

### RAG (Retrieval-Augmented Generation)
- Combines retrieval with generation
- Improves accuracy with external knowledge
- Reduces hallucinations
